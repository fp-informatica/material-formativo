# Material de FP Informática

Este repositorio contiene una colección de recursos útiles para estudiantes y profesionales interesados en las siguientes titulaciones:
- **Grado Medio**: Sistemas Microinformáticos y Redes (SMR)
- **Grado Superior**: Administración de Sistemas Informáticos en Red (ASIR)
- **Grado Superior**: Desarrollo de Aplicaciones Multiplataforma (DAM)
- **Grado Superior**: Desarrollo de Aplicaciones Web (DAW)
- **Curso de Especialización**: Ciberseguridad en Entornos de las Tecnologías de la Información (CETI)
- **Curso de Especialización**: Desarrollo de Videojuegos y Realidad Virtual (DVRV)
- **Curso de Especialización**: Inteligencia Artificial y Big Data (IABD)

Para más información sobre toda la oferta educativa de la familia profesional de Informática y Comunicaciones, revisa el siguiente enlace:

https://www.fp-informatica.es/oferta-educativa/
